FAQ

What is a Hald?

HALD stands for Hald CLUT, which is a 3D lookup table (LUT) that contains a grid of colours. It is used to apply a colour filter to an image. The HALD is a neutral image, meaning it has no colour information, and is used as a base for the LUT.

What is a LUT?

LUT stands for Look Up Table. It is a table that contains a set of data that is referenced by a software to apply a colour filter to an image. It is used to apply a colour filter to an image. The HALD is a neutral image, meaning it has no colour information, and is used as a base for the LUT.

How do I use a LUT?

Most image editing software supports LUTs. You can apply a LUT to an image by importing it into your software and applying it to your image. The LUT will apply the colour filter to your image.
